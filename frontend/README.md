# 考前刷题复习系统 — 前端

Vue 3 + Vite + Vue Router + Axios，对接 FastAPI 后端。

## 使用流程

```
登录 → 我的课程 → 创建课程 → 导入/手动加题 → 开始刷题 → 间隔复习
                      ↓                    ↓
                 发布 → 公共题库        错题本 → 错题复习
                 撤回 → 私有
```

1. **先创建课程**：进入"我的课程"创建一个课程（题库），例如"高等数学"
2. **导入或手动加题**：
   - 手动逐道添加（支持所有题型）
   - JSON 批量导入
   - Word/PPT 上传 → AI 预览 → 修改确认
3. **刷题**：选择课程后进入刷题模式，系统从该课程中随机抽题
4. **间隔复习**：每次答题后自动安排下次复习时间，在"到期复习"中集中回顾
5. **发布**：完成课程建设后可以发布到公共题库，所有用户共享
6. **撤回**：已发布的课程或题目可以随时撤回为私有

> 所有题目必须归属于某个课程。导入时未指定课程则自动归入"未分类题库"。

## Word/PPT 预览导入流程

```
上传文件 → AI 解析 → 预览题目列表 → 手动编辑 → 确认导入
                                                    ↓
                                              写入数据库
```

- 上传 Word（.docx）或 PPT（.pptx）文件
- 后端调用 AI 提取文本并解析为结构化题目
- 前端展示解析结果，允许用户修改题目内容
- 确认后统一写入数据库；确认前不写入任何数据

## 安装与启动

```powershell
cd frontend
npm.cmd install
npm.cmd run dev -- --host 0.0.0.0
```

> **Windows PowerShell 提示**：如果 `npm` 提示执行策略错误，请使用 `npm.cmd` 代替 `npm`，避免 npm.ps1 的执行策略问题。

电脑浏览器打开：

```text
http://127.0.0.1:5173
```

## 手机同 WiFi 访问

后端默认地址是：

```text
http://127.0.0.1:8000
```

如果手机访问前端，需要让前端调用电脑的局域网 IP。请在 `frontend/` 目录创建或修改 `.env`：

```env
VITE_API_BASE_URL=http://电脑局域网IP:8000
```

例如：

```env
VITE_API_BASE_URL=http://192.168.1.8:8000
```

然后重新启动前端：

```powershell
npm.cmd run dev -- --host 0.0.0.0
```

手机浏览器打开：

```text
http://电脑局域网IP:5173
```

## 功能

- 课程管理（创建、编辑、发布、撤回、删除我的课程）
- 手动逐道添加题目（支持单选、多选、判断、填空、简答）
- JSON 题目批量导入（归属到指定课程）
- Word/PPT 文件预览→编辑→确认导入
- AI 自动解析文档导入题目（需后端配置 API 密钥）
- 在课程内随机刷题、提交答案、显示对错
- 间隔复习（到期提醒、错题优先复习）
- 错题本查看和移除
- 练习统计和历史记录
- 公共题库浏览他人发布的课程和题目
- AI 对话助手
